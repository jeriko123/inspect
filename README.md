# 🐾 WildTrack — Мобильное приложение для учета диких животных

**WildTrack** — это легковесное мобильное приложение, разработанное на React Native (Expo) специально для инспекторов охраны природы. Оно позволяет оперативно фиксировать встречи с дикими животными в полевых условиях даже при полном отсутствии интернета. 

Накопленные данные надежно сохраняются на устройстве и экспортируются в формат CSV для последующего анализа или импорта в Google Таблицы.

---

## 🚀 Основные возможности

* **🌐 Offline-First архитектура:** Все данные локально записываются в реляционную базу данных SQLite. Приложение полностью автономно и не требует сети для работы в тайге, горах или заповедниках.
* **🧑‍✈️ Умный онбординг (Conditional Navigation):** Имя инспектора запрашивается один раз при первом запуске, сохраняется в системе и автоматически подставляется во все новые записи. Изменить его можно в любой момент в настройках.
* **📜 Оптимизированный список (Infinite Scroll):** Поддержка пагинации (загрузка порциями по 20 записей через `LIMIT` и `OFFSET`) гарантирует плавную работу интерфейса и быстрый скролл даже при десятках тысяч строк в базе.
* **📍 Геолокация в один клик:** Автоматический GPS-трекинг фиксирует точные географические координаты места встречи с животным.
* **📊 Экспорт в Google Таблицы (CSV):** Генерация CSV-файла прямо на устройстве и отправка через системное меню «Поделиться» (Google Диск, Telegram, почта и др.).
* **🔄 Импорт и восстановление:** Возможность быстро восстановить или объединить базу данных из ранее сохраненного CSV-файла.
* **🧹 Безопасная очистка:** Функция полного сброса локальной базы данных перед началом нового отчетного периода (с обязательным подтверждением действия, чтобы избежать случайной потери данных).

---

## 🛠 Стек технологий

* **Ядро:** React Native (Expo SDK 51+)
* **Стилизация:** NativeWind (Tailwind CSS v4) — адаптивный, современный и быстрый UI
* **Навигация:** React Navigation (Stack)
* **База данных:** `expo-sqlite` 
* **Стейт-менеджер:** Zustand + AsyncStorage (для персистентного хранения настроек пользователя)
* **Формы и валидация:** React Hook Form + Zod
* **Системные API:** * `expo-location` (GPS-координаты)
    * `expo-file-system` (работа с файловой системой устройства)
    * `expo-sharing` (шаринг отчетов)
    * `expo-document-picker` (импорт файлов)
* **UI Компоненты & Иконки:** `@react-native-community/datetimepicker`, `@react-native-picker/picker`, `lucide-react-native`

## 💻 Установка и запуск

### 1. Требования
Убедитесь, что у вас установлены **Node.js** (v18+) и менеджер пакетов (`npm`, `yarn` или `bun`).

### 2. Клонирование и установка зависимостей
```bash
# Клонируйте репозиторий
git clone [https://github.com/jeriko123/inspect.git](https://github.com/jeriko123/inspect.git)
cd inspect

# Установите зависимости
npm install

3. Запуск в режиме разработки
Bash
# Запуск Expo CLI
npx expo start
Тестирование на реальном устройстве: Установите приложение Expo Go на ваш Android/iOS девайс и отсканируйте QR-код из терминала.

Эмуляторы: Нажмите a для запуска на Android-эмуляторе или i для iOS-симулятора.

📊 Структура данных (Экспорт)
Сгенерированные CSV-файлы строго соответствуют концепции Tidy Data (одно наблюдение — одна строка).

Выгрузка содержит следующие столбцы:
Дата (YYYY-MM-DD) | Время (HH:MM) | Инспектор | Вид животного | Количество | Широта | Долгота | Примечания

💡 Совет: Данный формат идеально распознается Google Диском. При загрузке файла в облако выберите опцию «Открыть с помощью Google Таблиц» для автоматической конвертации в полноценную таблицу.
