WildTrack — Мобильное приложение для учета диких животных

WildTrack — это легковесное, мобильное приложение, разработанное на React Native (Expo). Оно предназначено для инспекторов охраны природы и позволяет оперативно фиксировать встречи с дикими животными в полевых условиях. Накопленные данные сохраняются локально и затем легко экспортируются в формат CSV для последующего импорта в Google Таблицы.

🚀 Основные возможности

Offline-First архитектура: Все данные надежно записываются в локальную реляционную базу данных SQLite.

Умный онбординг (Conditional Navigation): Приложение один раз запрашивает имя инспектора при первом запуске и автоматически подставляет его во все новые записи.

Оптимизированный список (Infinite Scroll): Поддержка пагинации (загрузка порциями по 20 записей с использованием LIMIT и OFFSET) гарантирует быструю работу интерфейса даже при десятках тысяч строк в базе.

Автоматический GPS-трекинг: Фиксация точных географических координат встречи с животным в один клик.

Экспорт в Google Таблицы (CSV): Генерация CSV-файла прямо на устройстве и отправка его через системное меню «Поделиться» (Google Диск, Telegram, Bluetooth и т.д.).

Импорт данных: Возможность восстановить всю базу из ранее сохраненного CSV-файла.

Безопасная очистка: Возможность сбросить базу данных перед началом нового отчетного месяца с подтверждением действия.

🛠 Стек технологий

Ядро: React Native (Expo SDK 51+)

Стилизация: NativeWind (Tailwind CSS v4) для быстрого и современного кастомного UI

Навигация: React Navigation (Stack)

База данных: expo-sqlite (новый синхронный API)

Стейт-менеджер: Zustand + AsyncStorage (для персистентного хранения настроек пользователя)

Формы и валидация: react-hook-form + zod

Системные API: expo-location (GPS), expo-file-system (файлы), expo-sharing (шаринг), expo-document-picker (выбор файлов)

UI Компоненты: @react-native-community/datetimepicker (дата/время), @react-native-picker/picker (селектор видов), lucide-react-native (иконки)

 

💻 Установка и запуск

1. Требования

Убедитесь, что у вас установлены Node.js (v18+) и менеджер пакетов (npm, yarn или bun).

2. Клонирование и установка зависимостей

# Клонируйте репозиторий
https://github.com/jeriko123/inspect.git
cd inspect

# Установите зависимости
npm install


3. Запуск в режиме разработки

# Запуск Expo CLI
npx expo start


Для тестирования на физическом устройстве установите приложение Expo Go на ваш Android/iOS девайс и отсканируйте QR-код из терминала.

Для запуска на эмуляторе Android нажмите a, на симуляторе iOS — i.

📊 Экспорт данных

Сгенерированные CSV-файлы соответствуют концепции Tidy Data (одно наблюдение — одна строка). Столбцы в выгрузке:

Дата (YYYY-MM-DD), Время (HH:MM), Инспектор, Вид животного, Количество, Широта, Долгота, Примечания

Этот формат идеально распознается сервисом Google Диск при загрузке и автоматически конвертируется в полноценные Google Таблицы.
